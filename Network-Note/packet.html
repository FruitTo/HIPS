Version (4 bits)
ระบุเวอร์ชันของ IP protocol ที่ใช้ (IPv4 = 4, IPv6 = 6)
IHL - Internet Header Length (4 bits)
ความยาวของ IP header หน่วยเป็น 32-bit words (ขั้นต่ำ 5, ส่วนใหญ่ใช้ 5)
Type of Service (8 bits)
กำหนดคุณภาพการบริการ (QoS) เช่น ความสำคัญ, ความเร็ว, ความน่าเชื่อถือ
Total Length (16 bits)
ความยาวรวมของ IP packet ทั้งหมด (header + data) หน่วยเป็น bytes
Identification (16 bits)
เลขประจำตัวของ packet เพื่อใช้ในการรวม fragments เข้าด้วยกัน
Flags (3 bits)
ควบคุมการแบ่งส่วน: Don't Fragment (DF), More Fragments (MF)
Fragment Offset (13 bits)
ตำแหน่งของ fragment นี้ในข้อมูลต้นฉบับ (หน่วย 8 bytes)
Time to Live (8 bits)
จำนวนครั้งสูงสุดที่ packet สามารถผ่าน router ได้ (ป้องกันการวนซ้ำ)
Protocol (8 bits)
ระบุ protocol ชั้นถัดไป (TCP=6, UDP=17, ICMP=1)
Header Checksum (16 bits)
ตรวจสอบความถูกต้องของ IP header (ไม่รวม data)
Source IP Address (32 bits)
ที่อยู่ IP ของผู้ส่ง
Destination IP Address (32 bits)
ที่อยู่ IP ของผู้รับ
